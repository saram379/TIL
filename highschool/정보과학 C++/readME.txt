#변수
 | C언어에서의 특징
    사용 전에 미리 선언 필수
    변수 선언 시 변수명, 자료형, 기억장소 크기 결정
    자료형과 자료가 일치해야함
    언제든 변경 가능
 | 변수명 작성 규칙
    영문자, 숫자, 밑줄, 공백X
    첫글자는 영문자 or 밑줄
    대,소문자 구분 엄격
    예약어 사용 불가
    
 | 자료형
    1 정수형(형 선언자(크기(byte), 값의 표현 범위))
      char(1, -2^7 ~ 2^7 -1), short(2, -2^15 ~ 2^15 -1), int(2, -2^31 ~ 2^31 -1), long(4, -2^31 ~ 2^31 -1), long long(8, -2^63 ~ 2^63 -1)
      각 자료형 앞에 "unsigned"를 붙이면 부호 없는 정수로 나타낼 수 있음(크기는 같고 표현범위가 모두 0부터 시작해 2^8-1, 2^16-1, ...)으로 변함
      자료범위보다 큰 값이 들어가면 오버플로된다.
    2 실수형
      float(4, -3.40e+38 ~ 3.40e+38), double(8, -1.79e+308 ~ 1.79e+308)
      대부분 IEEE754 표준에 따른다. (IEEE754:부동소수점을 표현하는 가장 널리 쓰이는 표준)
      *부동소수점(floating point)
      @@더 찾아보기!!
    3 문자형
      char(1, -2^7 ~ 2^7-1)
      문자 하나를 표현할 수 잇는 자료형
      아스키코드표상에 있는 문자를 표현
      
    - sizeof()
      기억장소의 크기를 바이트 단위로 구하는 연산자이다.
      e.g. printf("%d", sizeof(char)):   >>>  1
    - limits.h 헤더 파일
      자료형의 표현 범위에 대한 정보를 담고 있다
    - 2^7 = 128 / 2^15 = 32786 / 2^31 = 2147483648
  
 | 상수(constant)
    프로그램 실행 도중 값을 변경 불가한 데이터
    1 정수 상수
      소수점 포함 X
      수치 앞에 부호(+,-)포함 가능
      정수는 기본적으로 10진수, 수치 '앞'에 0을 붙이면 8진수, 0x를 붙이면 16진수로 인식
      정수 상수 기본 자료형: int(4byte, -2^31 ~ 2^31-1)
      수치 '뒤'에 LL or ll 붙이면 long long형 상수로 인식
      실습\ int constant.cpp 파일
        2147483648 + 1 의 결과가 -2147483638임 ==> int형의 표기 한계치인 2^31-1을 넘겨 최솟값인 -2^31이 저장됨
        2147483648LL + 1 의 결과가 2147483648임 ==> 산술연산에서 자료형이 다르면 표현 범위가 넓은 자료형에 맞춰 저장됨(여기서는 longlong형에 맞춰지기에 그대로 출력이 되는 것)
    2 실수 상수
      소수점을 사용, e, E를 사용해 지수형식으로 표현 가능
      수치 앞에 부호(+,-)포함 가능
      실수는 기본적으로 10진수로 인식
      기본 자료형은 double형(8byte), 수치 뒤에 F or f를 붙이면 float형 상수가 됨
       \실습 float constant.cpp 파일
         double로 상수를 잡으면 더 넓은 범위의 값을 입력 받을 수 있다
         %lf는 double형의 형식 지정자이다 / %f는 float, double의 형식 지정자이다. 따라서 %f하면 상관없이 쓸 수 있다!!
    3 문자 상수
       문자 하나를 작은따옴표로 둘러싸서 표현
       유니코드 문자, 아스키코드 문자 등 사용 가능
       이스케이프 시퀀스는 단일 문자로 간주
       문자 상수에 실제로 저장되는 것은 '문자 코드'
